투표 준비

1단계는 가장 당연하게도 투표 뽑기 프로그램을 실행하는 것 입니다.

2단계는 투표이름을 입력하세요!

3단계는 후보나 선택지를 생성버튼을 눌러 칸을 만든뒤에 이름을 써 넣으세요. 만약 칸을 너무 많이 만든경우 X 버튼을 눌러 칸을 지우세요. 선택지는 9개까지만 가능합니다.

4단계는 투표 시작을 누르는것입니다.

만약 모든 입력된 정보를 삭제하려면 리셋을 누르세요.

투표 방법

투표를 시작하면 이런 안내가 뜰것입니다.

  • 기권 : 0
  • 선택지 1 : 1
  • 선택지 2 : 2

각 번호에 해당되는 숫자키를 누르세요.

0.

표가 처리되면 안내창이 뜹니다. 안내창은 곧 자동으로 사라집니다. 안내창이 뜬 상태에서는 투표가 되지 않습니다! 창이 사라진뒤 누르세요

투표가 완료되면 5초후 공개합니다.

결과 보기

완료되면 표로 결과가 표시됩니다.

저장하려면 저장버튼을 누르고, 끝내려면 끝내기를 누릅니다.

먼저 모드를 추첨으로 전환합니다.

추첨명 (선턕)을 입력하고 후보 / 선택지 추가를 눌러 후보 / 선택지 란을 추가한뒤 입력합니다.
